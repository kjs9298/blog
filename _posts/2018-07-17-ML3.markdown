* L2 loss
```
 = the square of the difference between the label and the prediction
 = (observation - prediction(x))2

```  

* MSE (Mean square error)
$$MSE = \frac{1}{N} \sum_{(x,y)\in D} (y - prediction(x))^2$$
  * D = 데이터셋
  * N = example 의 수
